discord-mesaj-silici
====================

Discord DM ve grup sohbetlerinde kendi mesajlarınızı hızlıca silmek için sade bir **selfbot** projesi.

### Kurulum

1. Depoyu klasörünüze indirin / kopyalayın.
2. Bağımlılıkları yükleyin:

   ```bash
   npm install
   ```

3. Aynı klasörde bir `.env` dosyası oluşturun ve Discord token’ınızı ekleyin:

   ```env
   TOKEN=buraya_discord_tokeniniz
   ```

4. Botu başlatın:

   ```bash
   npm start
   ```

> Not: Tam komutlar ve kullanım şekli `index.js` içindeki yapılandırmanıza göre değişebilir.
